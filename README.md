# 星航抢晶战

这是一个可以直接在浏览器里玩的轻量对战小游戏。我先做成了一个小型试玩版：不用下载、不用注册，打开链接就能开一局。

试玩地址：

https://brillernet-droid.github.io/star-courier/

## 这是个什么游戏？

你会驾驶一台小型星航机体，在裂隙竞技场里和 AI 或朋友同屏对战。核心玩法很简单：抢能量晶体、躲子弹、找机会用技能反打，最后看谁能先完成目标。

目前它不是一个大型游戏，更像一个可以继续扩展的小原型。我现在主要想把手感、关卡、排名和对战节奏先跑通，后面可以继续加更多地图、道具、角色和联网玩法。

## 现在可以玩什么？

- 单人 AI：一个人也能玩，适合先熟悉操作。
- 双人同屏：两个人用同一台电脑对战。
- 三种机体：
  - 疾影：速度快，适合抢晶和突袭。
  - 堡垒：血量高，适合硬扛和守点。
  - 脉冲：比较均衡，技能偏控场。
- 四种玩法：
  - 抢晶：抢晶体得分，命中对手可以打落分数。
  - 闪斗：节奏更快，被击毁后会复活，适合快速对战。
  - 决斗：更像传统单挑，主要目标是击毁对手。
  - 占点：围绕中央区域争夺积分。
- 三个关卡：
  - 边境训练场
  - 陨石中继站
  - 核心裂隙

## 排名和奖励

游戏里有一个本地排名榜。P1 赢下关卡后，会根据得分、剩余时间、剩余血量和击毁奖励计算一个评分，并保留前 5 名。

这个排名目前保存在当前浏览器里，所以它更像“这台电脑上的小排行榜”。如果换浏览器或换设备，排名不会同步。

## 操作方式

P1：

- `WASD` 移动
- `Space` 射击
- `Shift` 技能
- 单人模式下也可以用方向键，或者拖动画面移动

P2：

- 方向键移动
- `Enter` 射击
- `/` 技能

其他：

- `P` 暂停

## 适合怎么试玩？

如果你第一次打开，可以先选：

1. 单人 AI
2. 抢晶
3. 疾影
4. 开战

熟悉之后再试“闪斗”，这个模式节奏更快，也更容易打出排名分。

## 当前版本说明

这个项目目前是一个前端小游戏原型，全部代码都在仓库里，没有后端服务，也没有构建步骤。打开网页就能玩。

后面可能会继续加：

- 更多关卡和地图机制
- 更多机体和技能
- 更完整的奖励系统
- 更好看的结算页面
- 真正的在线排行榜
- 手机端更舒服的操作

## 本地打开

如果你想在电脑上直接打开源码，可以双击 `index.html`。

也可以在项目目录里运行：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```
